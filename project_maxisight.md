Maxisight is open source CLI infrastructure for job automation — the Crawlee of job hunting. Free framework for developers, premium hosted product for non-technical users (Crawlee → Apify playbook).

**Mission:** Give broke, depressed, anxious job seekers maximum sight — a direction, hope. Not a feature, a mission.

**CLI:** `pip install maxisight` / `maxisight crawl / match / apply / outreach`

**Two-layer business model:**
- Layer 1: Open source CLI on PyPI (free) — builds stars, community, O-1 evidence
- Layer 2: Hosted cloud product (premium pricing) — non-technical users pay for removal of friction

**Target user Phase 1:** Developers comfortable with terminal and API keys
**Target user Phase 2:** Non-technical job seekers who pay premium

**Why:** People paying $200-300/month for tools that are just LLM prompts in a UI. ApplyPilot has 93% failure rate. No open source tool has cross-company job discovery or Pipeline B (funded startup outreach).

**How to apply:** Always keep the mission in mind when making architectural tradeoffs. Free, honest, deterministic, composable. Never add paid gates or subscriptions to the CLI.
